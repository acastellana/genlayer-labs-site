§3 — Research Arc

Five Systems,
One Thesis

Each system encodes a research question. The arc is coherent: minimum description length, maximum capability.

2021–2022
yAgents
Precursor · Self-Reflection

The first self-reflecting terminal agent — before Claude Code, before OpenClaw. A system that observed its own execution trace, reasoned over it, and modified behaviour in the next step. The kernel of autonomous agency.

Core insight: recursive self-observation as the minimal substrate of agency

2022–2023
GenWorlds
Multi-Agent Coordination

The first general framework for networks of autonomous LLM agents. Coordination without a central planner: shared world state, message-passing, emergent division of labour. The ancestor of every modern agent framework.

Core insight: emergent coordination from local interaction rules

2023–present
GenLayer
Flagship · Decision Infrastructure

Intelligent Contracts: programs that call LLMs to resolve ambiguous predicates, verified by a decentralised validator network. The formal mathematical treatment — published with CRM (Centre de Recerca Matemàtica, UAB) — constitutes original research in probabilistic agreement theory.

Core insight: subjective verification formalised as a CLT on Banach-valued random variables

2024
SubZeroClaw
Kolmogorov Minimum · 54 KB

A 54 KB autonomous agent runtime in C. LLMs already know Linux — therefore the minimum description length of an agentic loop is achieved by treating the shell as a native tool. One binary. One dependency. No frameworks.

Core insight: $K(\text{agent}) \approx K(\text{LLM}) + K(\text{shell interface}) + O(1)$ — everything else is accidental complexity

2025–present
SubZero-SWARM & Genetic Optimisation
Active Research

GenWorlds v2 built on the SubZeroClaw kernel. Swarm behaviour emergent from minimal agents. Genetic algorithms as a search procedure over swarm topologies — evolution as the outer optimisation loop over inner agent policies.

Core insight: swarm fitness as a learnable objective; topology as a genotype
